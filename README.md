# Athena

simple search library support T9

# How do I use?

1. Add it in your root build.gradle at the end of repositories:

```
allprojects {
     repositories {
         ...
         maven { url 'https://jitpack.io' }
     }
 }
```

1. Add the dependency

```
dependencies {
        compile 'com.github.User:Repo:Tag'
    }
```
